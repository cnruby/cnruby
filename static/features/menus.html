Multilayer Menus

if you want to create a menu that contains parent-child relationships, you just need to create folders. simpler-paper automatically generate menus based on hierarchical relationships. like:

**doc source**
|____docs
| |____install
| | |____install.1.md
| | |____install.2.md
| |____problems
| | |____answer.1.md
| | |____answer.2.md
| |____paper.config.json

run paper bd and get the following effect:



Ordering
